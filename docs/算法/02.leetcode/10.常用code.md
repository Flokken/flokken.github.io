## 常用代码

### 数字，列表，字符串等互转

假设有一个整数12345，我们常常可能需要转换他为各种形式来操作他

比如列表[1,2,3,4,5]，字符串="12345",char数组['1','2','3','4','5']。

```java
int num=12345;
//整数转列表
int tmp=num;
List<Integer>list = new ArrayList<>();
while(tmp!=0){
    list.add(0,tmp%10);
    //注意是首位，如果是直接list.add(tmp%10);就是倒序，list=[5,4,3,2,1];
    tmp/=10;
}
//list=[1,2,3,4,5]
```
