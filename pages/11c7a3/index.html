# 数字，列表，字符串等互转

假设有一个整数12345，我们常常可能需要转换他为各种形式来操作他

比如列表[1,2,3,4,5]，字符串="12345",char数组['1','2','3','4','5']。

int num=12345;
//整数转列表
int tmp=num;
List<Integer>list = new ArrayList<>();
while(tmp!=0){
    list.add(0,tmp%10);
    //注意是首位，如果是直接list.add(tmp%10);就是倒序，list=[5,4,3,2,1];
    tmp/=10;
}
//list=[1,2,3,4,5]

int []数组转List<Integer>

int[] intArray = {1, 2, 3, 4, 5};
List<Integer> integerList = IntStream.of(intArray).boxed().collect(Collectors.toList());
1
2

解释:

  • 使用 IntStream.of 方法将 intArray 转换为一个 IntStream 对象。

  • 接着，我们使用 boxed 方法(装箱)将 IntStream 中的每个基本类型 int 包装为对应的 Integer 包装类对象。

  • 最后，我们使用 collect 方法和 Collectors.toList() 收集器将流中的元素收集到一个新的 List<Integer> 对象中。

Integer [] 转List<Integer>

# 10进制转2进制

除2取余法，比如10， 10/2 =5， 0 5/2 = 2， 1（除不尽向下取整） 2/2= 1， 0 1/2=0, 1 ,所以结果是1010.

public int ten2two(int num){
    StringBuilder sb = new StringBuilder();
    while(num!=0){
        sb.append(num%2);//求余数
        num>>=1;//除以2
    }
    return sb.toString();
}

# 集合进行降序排序

集合排序是Collections.sort()，数组排序是Arrays.sort()

import java.util.ArrayList;
import java.util.Collections;
import java.util.Comparator;
import java.util.List;

public class Main {
    public static void main(String[] args) {
        List<Long> sum = new ArrayList<>();
        sum.add(5L);
        sum.add(2L);
        sum.add(8L);
        sum.add(1L);

        // 降序排序
        Collections.sort(sum, (num1, num2) -> Long.compare(num2, num1));

        // 输出排序后的结果
        for (Long num : sum) {
            System.out.println(num);
        }
    }
}

更常见的是重载比较器

import java.util.ArrayList;
import java.util.Collections;
import java.util.Comparator;
import java.util.List;

public class Main {
    public static void main(String[] args) {
        List<Long> sum = new ArrayList<>();
        sum.add(5L);
        sum.add(2L);
        sum.add(8L);
        sum.add(1L);

        // 降序排序
        Collections.sort(sum, new Comparator<Long>() {
            @Override
            public int compare(Long num1, Long num2) {
                if (num1 > num2) {
                    return -1;
                } else if (num1 < num2) {
                    return 1;
                } else {
                    return 0;
                }
            }
        });

        // 输出排序后的结果
        for (Long num : sum) {
            System.out.println(num);
        }
    }
}

Question:对象数组为什么还需要初始化其中的元素?

假设我们定义一个List数组，我们怎么初始化?

private List<Integer>[] array;
1

首先我们理解一下，这个array是一个数字，其元素类型是List<Integer>。所以可以如下初始化

array=new List[n];
//array=new ArrayList[n];
1
2

现在这样是创建了一个List对象数组，其元素都是null，所以还要对元素进行初始化

  • 因为new List[n]是动态初始化，而动态初始化时，元素类型如果是引用类型，那么默认值是null，所以还需要再对元素值进行初始化
 Arrays.setAll(array, e -> new ArrayList<>());
1

如果是基础类型数组比如int [][]array = new int[n][n]，这样也是动态初始化，但是int型默认初始值是0，所有当然不用后面那步给元素初始化
