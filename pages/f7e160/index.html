# 2588. 统计美丽子数组数目

2588. 统计美丽子数组数目 (opens new window)

思路：条件可以转化为求异或和为零的子数组，然后我们想到子数组和，所以可以用前缀异或和优化。

那么我们需要枚举每一个子数组，这样做是O(n2)。所以应该还存在优化。

注意到，假设子数组是，那么就是。那么。

此时可以枚举，然后利用哈希表记录，那么可以一次遍历求出答案。（这个优化思路很常用）

class Solution {
    public long beautifulSubarrays(int[] nums) {
//011 001 010
//找到异或值为0的子数组数目
        int n= nums.length;
        long ans=0;
        int []s = new int[n+1];//前缀异或和
        for(int i=0;i<n;i++){
            s[i+1]=s[i]^nums[i];
        }
        Map<Integer,Integer>mp = new HashMap<>();
        for(int x:s){
            ans+=mp.getOrDefault(x,0);
            mp.merge(x,1,Integer::sum);
        }
        return ans;
    }
}

# 1094. 拼车

1094. 拼车 (opens new window)

灵神题解：

https://leetcode.cn/problems/car-pooling/solutions/2550264/suan-fa-xiao-ke-tang-chai-fen-shu-zu-fu-9d4ra

其实就是维护一个差分数组，并使用差分数组进行区间操作，然后通过差分数组还原原数组

注意：这里的区间操作是，因为在已经下车了，所以是左开右闭区间

class Solution {
    public boolean carPooling(int[][] trips, int capacity) {
        int n=trips.length;
        int []d = new int[1001];
        for(int []t:trips){
            int num=t[0],from = t[1],to=t[2];
            d[from]+=num;
            d[to]-=num;
        }
        int s=0;
        for(int x:d){
            s+=x;
            if(s>capacity){
                return false;
            }
        }
        return true;
    }
}

# 1109. 航班预订统计

1109. 航班预订统计 (opens new window)

思路：

维护一个差分数组进行区间操作，最后还原为数组即可。注意​是闭区间

注意：由于这里查询从1开始到n，所以可以把差分数组也初始化到1-n

class Solution {
    public int[] corpFlightBookings(int[][] bookings, int n) {
        int []d =new int [n+2];//因为题目查询从1开始
        int []ans =new int [n];
        for(int []book:bookings){
            int x=book[0],y=book[1],num = book[2];
            d[x]+=num;
            d[y+1]-=num;
        }
        int s=0;
        for(int i=1;i<=n;i++){
            s+=d[i];
            ans[i-1]=s;
        }
        return ans;
    }
}

# 2381. 字母移位 II(学习，循环移位)

2381. 字母移位 II (opens new window)

思路：

这里是a-z循环移位，那么可以把向前移位看成-1，向后移位看成1.对于移位结果，我们都应该取模。

  • 0->-1
  • 1->1

那我们使用一个差分数组维护移位后的结果，然后计算这个移位后字符串即可

注意：

  • 对于负数，我们一直用，直到其变成一个正数

  • 假设移位数是,使用来计算移位后结果

class Solution {
    public String shiftingLetters(String s, int[][] shifts) {
        int mod =26;
        int n=s.length();
        int []d =new int[n+1];
        for(int i=0;i<shifts.length;i++){
            int []sh =shifts[i];
            int x = sh[0],y=sh[1],num= (sh[2]==0?-1:1);
            d[x]+=num;
            d[y+1]-=num;
        }
        StringBuilder sb = new StringBuilder();
        int sum=0;
        for(int i=0;i<n;i++){
            char ch =s.charAt(i);
            sum+=d[i];
            int step = (sum+mod*10000)%mod;//根据数据范围，我们这样做保证其是一个正数
            sb.append((char)('a'+(ch-'a'+step)%mod));
        }
        return sb.toString();
    }
}
