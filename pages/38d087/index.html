# жҰӮиҝ°

гҖҢе Ҷ heapгҖҚжҳҜдёҖз§Қж»Ўи¶ізү№е®ҡжқЎд»¶зҡ„е®Ңе…ЁдәҢеҸүж ‘пјҢдё»иҰҒеҸҜеҲҶдёәдёӨз§ҚпјҡеӨ§йЎ¶е Ҷе’Ңе°ҸйЎ¶е ҶгҖӮ

  • гҖҢеӨ§йЎ¶е Ҷ max heapгҖҚпјҡд»»ж„ҸиҠӮзӮ№зҡ„еҖј вүҘ е…¶еӯҗиҠӮзӮ№зҡ„еҖјгҖӮ
  • гҖҢе°ҸйЎ¶е Ҷ min heapгҖҚпјҡд»»ж„ҸиҠӮзӮ№зҡ„еҖј вүӨ е…¶еӯҗиҠӮзӮ№зҡ„еҖјгҖӮ

е ҶдҪңдёәе®Ңе…ЁдәҢеҸүж ‘зҡ„дёҖдёӘзү№дҫӢпјҢе…·жңүд»ҘдёӢзү№жҖ§гҖӮ

  • жңҖеә•еұӮиҠӮзӮ№йқ е·ҰеЎ«е……пјҢе…¶д»–еұӮзҡ„иҠӮзӮ№йғҪиў«еЎ«ж»ЎгҖӮ
  • жҲ‘д»¬е°ҶдәҢеҸүж ‘зҡ„ж №иҠӮзӮ№з§°дёәвҖңе ҶйЎ¶вҖқпјҢе°Ҷеә•еұӮжңҖйқ еҸізҡ„иҠӮзӮ№з§°дёәвҖңе Ҷеә•вҖқгҖӮ
  • еҜ№дәҺеӨ§йЎ¶е ҶпјҲе°ҸйЎ¶е ҶпјүпјҢе ҶйЎ¶е…ғзҙ пјҲеҚіж №иҠӮзӮ№пјүзҡ„еҖјеҲҶеҲ«жҳҜжңҖеӨ§пјҲжңҖе°Ҹпјүзҡ„гҖӮ

# е Ҷеёёз”Ёж“ҚдҪң

жҲ‘д»¬еңЁеҶҷд»Јз Ғж—¶пјҢз»ҸеёёиҜҙд»Җд№Ҳз”Ёдјҳе…ҲйҳҹеҲ—жқҘе®һзҺ°е ҶпјҢйңҖиҰҒжҢҮеҮәзҡ„жҳҜпјҢиҝҷжҳҜеӣ дёәи®ёеӨҡзј–зЁӢиҜӯиЁҖжҸҗдҫӣзҡ„жҳҜгҖҢдјҳе…ҲйҳҹеҲ— priority queueгҖҚпјҢиҝҷжҳҜдёҖз§ҚжҠҪиұЎж•°жҚ®з»“жһ„пјҢе®ҡд№үдёәе…·жңүдјҳе…Ҳзә§жҺ’еәҸзҡ„йҳҹеҲ—гҖӮе®һйҷ…дёҠпјҢе ҶйҖҡеёёз”ЁдҪңе®һзҺ°дјҳе…ҲйҳҹеҲ—пјҢеӨ§йЎ¶е ҶзӣёеҪ“дәҺе…ғзҙ жҢүд»ҺеӨ§еҲ°е°ҸйЎәеәҸеҮәйҳҹзҡ„дјҳе…ҲйҳҹеҲ—гҖӮ

д»ҺдҪҝз”Ёи§’еәҰжқҘзңӢпјҢжҲ‘д»¬еҸҜд»Ҙе°ҶвҖңдјҳе…ҲйҳҹеҲ—вҖқе’ҢвҖңе ҶвҖқзңӢдҪңзӯүд»·зҡ„ж•°жҚ®з»“жһ„гҖӮ

еӣ жӯӨеҸҜд»ҘдёҚеҜ№дёӨиҖ…дёҚеҒҡзү№еҲ«еҢәеҲҶпјҢз»ҹдёҖеҸ«еҒҡвҖңе ҶвҖң

е Ҷеёёз”Ёзҡ„ж“ҚдҪңе’Ңж ҲпјҢйҳҹеҲ—еҫҲеғҸпјҢеҚі

  • е…Ҙе ҶпјҢеҮәе ҶпјҢеӨ§е°ҸпјҢжҹҘзңӢе ҶйЎ¶пјҢжҹҘзңӢе ҶжҳҜеҗҰдёәз©ә

# жҺ’еәҸи§„еҲҷйҮҚиҪҪ

еҰӮжһңдёҚжҳҜдјҳе…ҲйҳҹеҲ—пјҢжҜ”иҫғеҷЁз”ЁдәҺеҲ—иЎЁж—¶

  • return o1-o2иЎЁзӨәиҫғе°Ҹе…ғзҙ дјҳе…Ҳзә§жӣҙй«ҳпјҢеҚҮеәҸжҺ’еәҸ
  • return o2-o1иЎЁзӨәиҫғеӨ§е…ғзҙ дјҳе…Ҳзә§й«ҳпјҢйҷҚеәҸжҺ’еәҸ

еӣ дёәжҜ”иҫғеҷЁзҡ„compareжҺ’еәҸж—¶пјҢи§„еҲҷеҰӮдёӢпјҡ

  1. еҰӮжһң o1 е°ҸдәҺ o2пјҢеҲҷ o1 - o2 зҡ„з»“жһңжҳҜиҙҹж•°пјҢеӣ жӯӨдјҡиҝ”еӣһдёҖдёӘиҙҹж•ҙж•°пјҢиЎЁзӨә o1 еңЁжҺ’еәҸдёӯеә”иҜҘжҺ’еңЁ o2 зҡ„еүҚйқўгҖӮ
  2. еҰӮжһң o1 зӯүдәҺ o2пјҢеҲҷ o1 - o2 зҡ„з»“жһңжҳҜ0пјҢеӣ жӯӨдјҡиҝ”еӣһ0пјҢиЎЁзӨә o1 е’Ң o2 зҡ„йЎәеәҸдҝқжҢҒдёҚеҸҳгҖӮ
  3. еҰӮжһң o1 еӨ§дәҺ o2пјҢеҲҷ o1 - o2 зҡ„з»“жһңжҳҜжӯЈж•°пјҢеӣ жӯӨдјҡиҝ”еӣһдёҖдёӘжӯЈж•ҙж•°пјҢиЎЁзӨә o1 еңЁжҺ’еәҸдёӯеә”иҜҘжҺ’еңЁ o2 зҡ„еҗҺйқўгҖӮ

еӣ жӯӨпјҢдҪҝз”Ё return o1 - o2 зҡ„жҜ”иҫғи§„еҲҷеҸҜд»Ҙе®һзҺ°еҚҮеәҸжҺ’еәҸпјҢе…¶дёӯиҫғе°Ҹзҡ„еҖјдјҡжҺ’еңЁеүҚйқўпјҢиҫғеӨ§зҡ„еҖјдјҡжҺ’еңЁеҗҺйқўгҖӮ

жүҖд»ҘиҰҒжғіе®һзҺ°йҷҚеәҸжҺ’еәҸпјҢеҸҚиҝҮжқҘеҚіеҸҜгҖӮ

# Top-Kй—®йўҳ

Q:з»ҷе®ҡдёҖдёӘй•ҝеәҰдёә n ж— еәҸж•°з»„ nums пјҢиҜ·иҝ”еӣһж•°з»„дёӯеүҚ k еӨ§зҡ„е…ғзҙ гҖӮ

жңҖеҘҪжғізҡ„е°ұжҳҜжҺ’еәҸпјҢжҺ’еәҸдёҖиҲ¬жҳҜO(nlogn)

еҹәдәҺе ҶеҸҜд»Ҙжӣҙй«ҳж•Ҳзҡ„и§ЈеҶіиҜҘй—®йўҳгҖӮ

жіЁж„ҸпјҡеүҚKеӨ§жҳҜе»әз«Ӣе°Ҹж №е ҶпјҢеүҚKе°ҸжҳҜе»әз«ӢеӨ§ж №е ҶгҖӮ

д»”з»ҶзңӢдёӢйқўзҡ„з®—жі•жөҒзЁӢпјҢи§Јзӯ”дәҶжҲ‘еӨҡе№ҙзҡ„з–‘жғ‘

жұӮеүҚKеӨ§жҖқи·Ҝпјҡ

  1. еҲқе§ӢеҢ–дёҖдёӘе°ҸйЎ¶е ҶпјҢе…¶е ҶйЎ¶е…ғзҙ жңҖе°ҸгҖӮ

class Solution {
    public int[] topKFrequent(int[] nums, int k) {
        Map<Integer,Integer>map  =new HashMap<>();
        for(int num:nums){
            if(map.containsKey(num)){
                map.put(num,map.get(num)+1);
            }else{
                map.put(num,1);
            }
        }//е°Ҹж №е ҶжұӮеүҚkеӨ§
        PriorityQueue<Integer>pq = new PriorityQueue<>(new Comparator<Integer>(){
            @Override
            public int compare(Integer o1,Integer o2){
                return map.get(o1)-map.get(o2);
            }
        });
        for(Integer key:map.keySet()){
            if(pq.size()<k){
                pq.add(key);
            }else if(map.get(key)>map.get(pq.peek())){
                pq.remove();
                pq.add(key);
            }
        }
       int n=nums.length;
       int []res = new int [k];
       int i=0;
        while(!pq.isEmpty()){
            res[i]=pq.remove();
            i++;
        }
        return res;
    }
}
