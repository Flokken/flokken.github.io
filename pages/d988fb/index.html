# 理论学习

我们通过二叉树来学习递归，因为二叉树就是递归定义的。

求二叉树的最大深度?

104. 二叉树的最大深度 (opens new window)

# 经验：

二叉树不要一上来就去关注二叉树细节，而是思考整棵树和左右子树的关系

这个想法非常重要，递归不能去迷失到细节，而是搞清楚递归关系。

# 思考过程

我们把左右子树看成三角形，整棵树的最大深度 = max(左子树最大深度，右子树最大深度)+1(递推式)

而对于左右子树，其最大深度可以用一样的方法求得

原问题: 计算整棵树最大深度

子问题: 计算左右子树的最大深度

原问题和子问题可以用相似的方法求出

类比循环，既然过程相似，那么代码也应该基本相同，但是子问题需要把计算结果返回给上一级问题，所以更适合用递归实现

由于子问题规模比原问题小，不断递下去，总会到达尽头，即递归的边界条件(base case)，直接返回他的答案(归)

递在程序执行时，就是不断调用这个函数的过程，归就是达到边界返回结果到上一层栈帧的过程

边界条件

对于这道题，边界显然就是左子树或者右子树为空，那么此时返回0(该子树深度)

递归的正确性

主要思想是数学归纳法。数学归纳法 (opens new window)

递归如何执行

总的来说就是系统执行递归时，只要递归没有返回，那么这一次执行的栈帧是会保留的，知道最终的调用结束，把结果一级级的返回。

我们需要这种先进后出的数据结构，显然就是栈

# 维护全局变量的递归思路

上面的递归思路是不断返回上一层计算结果，最后返回最大深度

还有一种思路就是我们可以把当前的节点深度传入下一层，维护一个全局变量ans，然后在node==null的时候，更新这个ans。

class Solution {
    int ans=0;
    public void dfs(TreeNode root,int cur) {
        if(root == null){
            ans = Math.max(ans,cur);
            return;
        }
        dfs(root.left,cur+1);
        dfs(root.right,cur+1);
    }
    public int maxDepth(TreeNode root) {
        dfs(root,0);
        return ans;
    } 
}

# 111. 二叉树的最小深度

111. 二叉树的最小深度 (opens new window)

思路：

根据题意，对于一棵树而言，其最小深度就是左子树和右子树深度中最小的深度加1

既可以在归的过程中求，也可以在递的过程中求

在[归]的过程中求（无需全局变量）

分类讨论：

  • 如果当前节点为空，返回0
  • 如果当前节点只有左儿子，那么其左儿子最小深度应该取左子树找
  • 如果当前节点有只有右儿子，那么其右儿子最小深度应该去其右儿子找
  • 如果都有则是两个取小的

应该使用后序遍历，因为我们要知道了左右儿子返回的最小深度才能返回

class Solution {
    public int minDepth(TreeNode root) {
        //这里在归的过程中求
        return dfs(root);
    }
    public int dfs(TreeNode root){
        if(root==null) return 0;
        int left=dfs(root.left);
        int right = dfs(root.right);
        if(left!=0&&right==0) return left+1;
        if(right!=0&&left==0) return right+1;
        return Math.min(left,right)+1;
    }
}

在[递]的过程中求；

  • 由于我们传入的d，所以我们应该使用前序遍历，当遇到叶子节点时，就更新一次最小深度。
class Solution {
    int ans=Integer.MAX_VALUE;
    public int minDepth(TreeNode root) {
        dfs(root,1);
        return ans==Integer.MAX_VALUE?0:ans;
    }
    public void dfs(TreeNode root,int d){
        if(root==null){
            return ;
        }
        if(root.left==null&&root.right==null){
            ans=Math.min(d,ans);
            return;
        }
        dfs(root.left,d+1);
        dfs(root.right,d+1);
    }
}

# 112. 路径总和

112. 路径总和 (opens new window)

思路：

目标为,那么在根节点，就变成了去其子树中找到叶子节点和为的和。

边界：

  • 那么当并且当前节点是叶子节点时，说明有，返回true

  • 如果递归到null，返回false

注意

  • 这里注意到可能本来就是负数，所以小于0不用返回
    • 如果都是正数才可以提前返回
  • 必须终点是叶子节点才算是路径和

不使用全局变量

class Solution {
    public boolean hasPathSum(TreeNode root, int targetSum) {
        if(root==null) return false;
        return dfs(root,targetSum);
    }
    public boolean dfs(TreeNode root,int targetSum){
        if(root==null){
            return false;
        }
        if(targetSum-root.val==0 &&root.left==null&&root.right==null){
            return true;
        }
        return dfs(root.left,targetSum-root.val)||dfs(root.right,targetSum-root.val);
    }
}

# 113. 路径总和 II

113. 路径总和 II (opens new window)

思路：

和上一题基本一样，到达叶子节点并且和刚好为零时是一条路径，

区别在于需要记录答案，这里可以用回溯，维护一个LIst记录路径即可

边界条件

  • 如果当前为空，直接返回
  • 如果当前是叶子节点，判断是否为零，记录答案
    • 这里也可以不返回，因为叶子节点后面就是空节点，马上就会返回
class Solution {
    private List<List<Integer>>ans;
    private List<Integer>path;
    public List<List<Integer>> pathSum(TreeNode root, int targetSum) {
        ans =new ArrayList<>();
        path = new ArrayList<>();
        dfs(root,targetSum);
        return ans;
    }
    public void dfs(TreeNode root, int targetSum){
        if(root==null){
            return;
        }
        if(targetSum-root.val==0&&root.left==null&&root.right==null){
            path.add(root.val);
            ans.add(new ArrayList(path));
            path.remove(path.size()-1);
            return;
        }
        path.add(root.val);
        dfs(root.left,targetSum-root.val);
        path.remove(path.size()-1);

        path.add(root.val);
        dfs(root.right,targetSum-root.val);
        path.remove(path.size()-1);

    }
}

# 129. 求根节点到叶节点数字之和

129. 求根节点到叶节点数字之和 (opens new window)

树深度不超过10，节点值为[0,9]，所以肯定不会爆int

思路：

  • 维护一个和表示前缀和，每次到达叶子节点的时候记录和即可，所以仍然是回溯
  • 可以维护一个全局变量记录总和

边界

  • 空节点直接返回
  • 叶子节点可以返回也可以不返回，因为叶子节点后面就是空节点，马上也会返回
class Solution {
    int ans=0;
    int tmp=0;
    public int sumNumbers(TreeNode root) {
        dfs(root);
        return ans;
    }
    public void dfs(TreeNode root){
        if(root==null){
            return ;
        }
        tmp=tmp*10+root.val;
        if(root.left==null&&root.right==null){
            ans+=tmp;
        }
        dfs(root.left);
        dfs(root.right);
        tmp/=10;//相当于回溯
    }
}

# 257. 二叉树的所有路径

257. 二叉树的所有路径 (opens new window)

思路：

  • 和前面找路径的题基本一致，使用回溯来做，遇到叶子节点记录一次答案
class Solution {
    List<String>ans;
    List<Integer>path;
    public List<String> binaryTreePaths(TreeNode root) {
        path=new ArrayList<>();
        ans = new ArrayList<>();
        dfs(root);
        return ans;
    }
    public void dfs(TreeNode root){
        if(root==null) return;
        path.add(root.val);
        if(root.left==null&&root.right==null){
            StringBuilder sb= new StringBuilder();
            for(int i=0;i<path.size()-1;i++){
                sb.append(path.get(i));
                sb.append("->");
            }
            sb.append(path.get(path.size()-1));
            ans.add(sb.toString());
        }
        dfs(root.left);
        dfs(root.right);
        path.remove(path.size()-1);

    }

}

# 1448. 统计二叉树中好节点的数目

1448. 统计二叉树中好节点的数目 (opens new window)

思路：

  • 对于二叉树来说，可以把这个二叉树都是一条条的路径，直到叶子节点。所以可以维护这个路径上节点序列和一个最大值，如果最大值有变动，可以再求出最大值。
  • 如果当前元素是好节点，需要更新最大值，如果移除最大值，需要更新最大值

使用全局变量的写法

class Solution {
    List<Integer>path;
    int sum=0;
    int max=0;
    public int goodNodes(TreeNode root) {
        max=root.val;
        path=new ArrayList<>();
        dfs(root);
        return sum;
    }
    public void dfs(TreeNode root){
        if(root==null) return;
        path.add(root.val);
        if(root.val>=max) {
            sum+=1;
            max=getMax();
        }
        dfs(root.left);
        dfs(root.right);

        int last = path.get(path.size()-1);
        path.remove(path.size()-1);
        if(last==max) max=getMax();
    }

    public int getMax(){
        int max=-1000000;
        for(var num:path){
            max=Math.max(num,max);
        }
        return max;
    }
}

不使用全局变量的写法。

可以维护一个，表示到当前节点之前的最大值，每一层都向下传递这个最大值即可。

注意最后的三目运算符需要用括号包起来

class Solution {
    public int goodNodes(TreeNode root) {
        return dfs(root,-100000);
    }
    public int dfs(TreeNode root,int mx){
        if(root==null) return 0;
        if(root.val>=mx) {
            mx=root.val;
        }
        int left=dfs(root.left,mx);
        int right=dfs(root.right,mx);
        return left+right+ (root.val>=mx?1:0);
    }
}

# 987. 二叉树的垂序遍历

987. 二叉树的垂序遍历 (opens new window)

思路：

首先分析一下题目，要确定节点是那一列的，就需要给节点进行横纵坐标的编号。

那么可以搞出这样的三元组，将一样的节点放入一个数组，然后进行根据​进行排序

这道题不使用全局变量，每层传递上一个节点的更简单

  • 左儿子，右儿子​

如何实现？

我们把可以用一个hash表存储和。也就是根据把当成hash表的,二元组当成hash表的val。

因为可能重复并且需要按照col升序排列，所以这里使用有序集合TreeMap

TreeMap默认按照key进行升序排列

一遍dfs得到所有节点信息，只需要对二元组进行排序即可

  • 先按row升序排序，如果相等按升序排列
class Solution {
    public List<List<Integer>> verticalTraversal(TreeNode root) {
        Map<Integer,List<int[]>>groups = new TreeMap<>();
        dfs(root,0,0,groups);
        List<List<Integer>> ans=new ArrayList<>();
        for(List<int []>g:groups.values()){
            g.sort((a,b)->a[0]!=b[0]?a[0]-b[0]:a[1]-b[1]);//这个lambda表达式可以学习
            List<Integer>vals = new ArrayList<>();
            for(int[] p:g){
                vals.add(p[1]);//只要val
            }
            ans.add(vals);
        }
        return ans;
    }
    public void dfs(TreeNode root,int row,int col,Map<Integer,List<int[]>>groups){
        if(root==null) return;
        groups.computeIfAbsent(col,k->new ArrayList<>()).add(new int[]{row,root.val});
        dfs(root.left,row+1,col-1,groups);
        dfs(root.right,row+1,col+1,groups);
    }
}

# 总结

  • 递归不要去想具体子问题如何执行，我们需要想的是递推式以及边界条件。

    • 要找递推式，首先要能把原问题缩小为规模更小的子问题
    • 根据子问题，再去找边界条件

  • 递归两种写法

    • 一种是传递参数，使用全局变量维护答案
    • 一种是直接递归函数返回答案

  • 套路

    • 递归不要过于关注每一层怎么调用的，因为很复杂，只需要想好如何递归以及边界，那么根据数学归纳法，这个递归就是正确的

    • 递归调用时的栈帧

    • 递归的栈帧保存了这个调用时的状态（主要就是此次的各个参数）

    • 注意引用类型，虽然各自保存了栈帧，但是实际上还是指向同一空间

# 100. 相同的树

100. 相同的树 (opens new window)

原问题：二叉树当前节点相同，左子树相同，右子树也要相同

子问题：p左子树和q左子树相同，p右子树和q右子树相同

边界条件:

  • 如果有一个值不相同，那么就不相同，可以返回了
  • 如果子树或者节点是空时，需要判断一下另一个子树这了是不是空，如果是返回true，否则应该是false

维护一个全局变量:

维护一个全局变量ans，递归时传递当前状态给下一层

class Solution {
    boolean flag;
    public boolean isSameTree(TreeNode p, TreeNode q) {
        flag=true;
        is(p,q);
        return flag;
    }
    public void is(TreeNode p, TreeNode q){
        if(p==null&&q==null){
            return ;
        }
        if((p==null&&q!=null)||(p!=null&&q==null)||p.val!=q.val){
            flag=false;
            return;
        }
        is(p.left,q.left);
        is(p.right,q.right);
    }
}

class Solution {
    public boolean isSameTree(TreeNode p, TreeNode q) {
        if(p==null||q==null){
            return p==q;//如果都为空，返回true，否则应该是false（不相同）
        }
        if(p.val!=q.val){
            return false;
        }
        return isSameTree(p.left,q.left)&&isSameTree(p.right,q.right);

# 101. 对称二叉树

101. 对称二叉树 (opens new window)

这道题就在100题的基础上稍微改动一下。判断树是否轴对称

对于根节点，已经轴对称不用管。

对于轴对称，我们要用当前节点的左儿子和右儿子相比较。

  • 所以这里我们应该传入当前节点的左儿子和右儿子

边界问题

  • 如果有子树为空，比较另一个是否为空，返回
  • 左右节点值不相同，直接返回

原问题:根节点的左右子树根据轴对称

子问题:左子树根节点的val等于右子树节点的val，并且左儿子的子树和右儿子对称

轴对称就是左子树的左儿子等于右子树左儿子，左子树右儿子等于右子树左儿子。

注意：时刻牢记，我们这里的递归子问题，我们只比当前节点的val，没有涉及子树的val，那是下一层递归去比的

不维护全局变量

递归直接返回值

注意p，q可能为null的情况，因为不能对null访问val

  • 其中p,q一个为null，而另一个不为null，应该直接返回
  • 所以可以直接在访问val前，当p或q为null时判断一次是否p==q
class Solution {
    public boolean isSymmetric(TreeNode root) {
        return dfs(root.left,root.right);
    }
    public boolean dfs(TreeNode p,TreeNode q){
        if(p==null||q==null){
            return p==q;
        }
        if(p.val!=q.val) return false;
        return  dfs(p.left,q.right)&&dfs(p.right,q.left);
    }
}

# 110. 平衡二叉树

110. 平衡二叉树 (opens new window)

二叉树高度 = Max(左子树高度，右子树高度)+1

原问题:二叉树任意节点的左右子树的高度差不超过1

子问题:当前节点左右子树高度差不超过1，并且左右子树的内部的高度差也不会超过1

我们可以稍微改造一下之前求高度的代码。

边界条件

  • 当该节点左右子树高度差超过1，可以用可以每一层都返回来表示不平衡
    • 这样需要特判如果子树返回-1，就直接返回-1不再递归下去，每一层都返回-1回去，因为我们函数仍然是求高度
  • 当前子树返回-1，那么也直接返回-1

不使用全局变量

定义一个递归函数求高，如果左右子树高度差超过1，就返回-1。

最后判断最终返回时否为-1.

这样的好处是一旦遇到为-1的情况，就会一直返回，停止递归。

class Solution {
    public boolean isBalanced(TreeNode root) {
        return dfs(root)==-1?false:true;
    }
    public int dfs(TreeNode root){
        if(root==null) return 0;
        int L=dfs(root.left);
        if(L==-1) return -1;//这里必须截断否则覆盖了这个-1
        int R=dfs(root.right);
        if(R==-1) return -1;
        if(Math.abs(L-R)>1) return -1;
        return Math.max(dfs(root.left),dfs(root.right))+1;
    }
}

# 199.二叉树的右视图

199. 二叉树的右视图 (opens new window)

原问题:二叉树每一层的最右边的节点

子问题：在每一层取最右边节点

思路：

我们知道，前序遍历每次都是先去找到每一层最左边节点，所以我们可以改一下顺序，每次先递归右子树，那么每次遇到的节点就肯定是每一层最右边节点。

这道题我们可以先遍历右子树，这样就能保证先遇到右子树右边的节点。

问题

  • 怎么判断节点入队或者说这是新的一层?
    • 我们发现，维护一个序列ans，当ans长度为树的深度时，这个节点就可以加入（因为我们是从右开始遍历）。
    • 更细致的想，其实这就是在递的过程中，把节点都加入了，而归的过程中什么也不用干。

因为我们是在递的过程中判断当前节点是否加入ans，所以需要传入深度到下一次递归(区别于求高度，求高度是在归的过程中求)

边界：

  • 如果子树为空直接返回

这道题显然更适合使用全局变量记录答案

class Solution {
    List<Integer>ans;
    public List<Integer> rightSideView(TreeNode root) {
        ans=new ArrayList<>();
        dfs(root,0);//假设树根深度为0，深度==ans.size()时，需要加入答案序列
        return ans;
    }
    public void dfs(TreeNode root,int cur){
        if(root==null) return ;
        if(cur==ans.size()){
            ans.add(root.val);
        }
        dfs(root.right,cur+1);
        dfs(root.left,cur+1);
    }
}

# 26. 翻转二叉树

226. 翻转二叉树 (opens new window)

思路：

正确想法：

直接交换当前节点左右儿子节点即可达到翻转效果

所以应该递归当前节点

  • 需要处理当前节点左右儿子翻转

错误想法：卡了半小时

翻转就是将当前节点的左右子树关于轴对称的节点的值相互交换（错误了！这里没提轴对称！，每个节点的左右儿子都互换就能达到轴对称效果），所以应该分别递归当前节点的左右子树

互相交换当前的值如果当前有子树为空，需要特殊处理一下

而且这里需要处理根节点和左右儿子关系，所以肯定不能是递归两个儿子到下一层

边界

  • 如果当前节点为空，返回
class Solution {
    public TreeNode invertTree(TreeNode root) {
        dfs(root);
        return root;
    }
    public void dfs(TreeNode root){
       if(root==null) return;
        //假设root.right指向0x1
        //root.left指向0x2
        TreeNode tmp=root.right;//这里辨析一下引用
        //tmp也指向0x1
        //root.right指向0x2
        root.right=root.left;
        //root.left指向0x1
        root.left=tmp;
        dfs(root.left);
        dfs(root.right);
    }
}

# 1080. 根到叶路径上的不足节点

1080. 根到叶路径上的不足节点 (opens new window)

灵神题解：https://leetcode.cn/problems/insufficient-nodes-in-root-to-leaf-paths/solutions/2278769/jian-ji-xie-fa-diao-yong-zi-shen-pythonj-64lf

思路：

  • 对于叶子节点，其到根节点只有一条路径，所以如果其路径和<，这个节点需要删除
  • 对于非叶子节点，如果其左右儿子都被删除，那么这个节点也要删除
    • 因为如果一个中间节点左右儿子都被删除，说明到这个点的路径都是也是不足的，所以这个点也要删。

边界

  • 如果是空，直接返回

细节：

  • 不用维护表示前缀和，而是每次减去当前值，如果到了叶子节点，减去值后仍然>0，说明是不足节点
  • 注意这种题目要求修改二叉树的，需要直接修改root和左右子树

不用维护全局变量，每次传入该层

class Solution {
    public TreeNode sufficientSubset(TreeNode root, int limit) {
       return dfs(root,limit);
    }
    public TreeNode dfs(TreeNode root,int limit){
        if(root==null){
            return null;
        }
        limit-=root.val;
        if(root.left==null&&root.right==null){
            return limit>0?null:root;
        }//叶子节点

        //非叶子节点
        root.left=dfs(root.left,limit);
        root.right = dfs(root.right,limit);
        return root.left==null&&root.right==null?root=null:root;
    }
}

# 1026.节点与其祖先之间的最大差值（重要）

1026. 节点与其祖先之间的最大差值 (opens new window)

思路：

有一点很容易想到，一条路径中，B是A的子孙节点，那么B就是A的子孙，A是祖先。

由于我们是自上而下遍历，所以在从根节点到叶子节点的一条路径中，一定是先出现的是祖先，后面的是子孙

一开始做到这道题完全不会做，所以建议多看几遍灵神题解。

https://leetcode.cn/problems/maximum-difference-between-node-and-ancestor/solutions/2232367/liang-chong-fang-fa-zi-ding-xiang-xia-zi-wj9v

转换：

有了上面的思考，我们就可以想到，我们在自上而下的[递]的过程中，记录一条路径中的最大值和最小值，每个节点值与他们做差值，就可以得到最大差值（可以再优化，看后面）

在递的过程中寻求答案

class Solution {
    private int ans;
    public int maxAncestorDiff(TreeNode root) {
        dfs(root,root.val,root.val);
        return ans;
    }
    public void dfs(TreeNode root,int mn,int mx){
        if(root==null) return;
        ans=Math.max(ans,Math.max(root.val-mn,mx-root.val));
        dfs(root.left,Math.min(mn,root.val),Math.max(mx,root.val));
        dfs(root.right,Math.min(mn,root.val),Math.max(mx,root.val));
    }
}

在归的过程中寻找答案

class Solution {
    private int ans;

    public int maxAncestorDiff(TreeNode root) {
        dfs(root);
        return ans;
    }

    private int[] dfs(TreeNode node) {
        if (node == null) // 需要保证空节点不影响 mn 和 mx
            return new int[]{Integer.MAX_VALUE, Integer.MIN_VALUE};
        int mn = node.val, mx = mn;
        var p = dfs(node.left);
        var q = dfs(node.right);
        mn = Math.min(mn, Math.min(p[0], q[0]));
        mx = Math.max(mx, Math.max(p[1], q[1]));
        ans = Math.max(ans, Math.max(node.val - mn, mx - node.val));
        return new int[]{mn, mx};
    }
}

# 1372. 二叉树中的最长交错路径

1372. 二叉树中的最长交错路径 (opens new window)

独立完成一遍过，坚持下去，沉下心，一定会有收获！ 2024/04/02

思路：

在[递]的过程中寻找最长交错路径。

每次维护一个方向和当前交错路径长度，如果当前方向和上一个相反，长度置0；否则长度加1。

那么每次进入一个节点应该先更新答案。

递归结构

  • 对于左右节点都传入方向和当前路径长
    • 可以用一个变量记录方向，false表示左边，true表示右边
  • 可以分别从根节点左子树和右子树开始进行一次dfs

边界

  • 如果当前结点为空，返回。
class Solution {
    private int ans;
    public int longestZigZag(TreeNode root) {
        dfs(root.left,false,0);
        dfs(root.right,true,0);
        return ans;
    }
    public void dfs(TreeNode root, boolean dir,int sum){
        ans=Math.max(ans,sum);
        if(root==null) return;
        if(dir==true){
            dfs(root.left,false,sum+1);
            dfs(root.right,true,0);
        }else{
            dfs(root.left,false,0);
            dfs(root.right,true,sum+1);
        }
    }
}

# 总结

  • 回溯与递归的区别

    • 回溯是一种算法，而递归是一种编程技巧。递归可以用来实现回溯
    • 一般我们需要记录答案状态，比如二叉树所有路径，这就需要使用回溯
    • 但是统计路径数目就不需要使用回溯，直接递归就行

  • 做题模板？

    • 可以分为[递]和[归]两个过程，有些题，适合在递的过程中求解，有些适合归来求解

  • 使用哪种写法？

    • 看题目，有些适合直接传入变量，有些适合维护全局变量
    • 写法只是编程方不方便，并不实际影响

  • 常见题型

    • 如果要求修改二叉树，那么一般需要传入根节点，因为需要修改当前节点和其左右子树
    • 如果要记录答案序列，一般要维护一个全局变量
